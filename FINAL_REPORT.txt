📞 TROUBLESHOOTING:

  Issue: "Unable to load stores" message
    → Check: Backend running on http://localhost:8080
    → Check: Browser console (F12) for error details
    → Read: DEBUGGING_GUIDE.md

  Issue: CORS error in console
    → Already fixed! Rebuild if still happening: mvn clean install

  Issue: Backend won't start
    → Check: Java installed (java -version)
    → Check: Port 8080 not in use
    → Read: DEBUGGING_GUIDE.md → Backend section

  Issue: Frontend won't start
    → Check: Node.js installed (node --version)
    → Check: Port 4200 not in use
    → Run: npm install
    → Read: DEBUGGING_GUIDE.md → Frontend section
